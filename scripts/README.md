# Welcome to VibeOS

This is a **demonstration** of the Markdown Viewer!

## Features

The viewer supports:

- Headers (H1, H2, H3)
- Bullet lists
- Horizontal rules
- Code blocks
- Word wrapping

---

## Code Example

Here is some code:

```
def hello():
    print("Hello VibeOS!")
    return 42
```

Pretty cool, right?

### Controls

- Use **Up/Down** arrows to scroll
- Use **Page Up/Down** for faster scrolling
- Press **q** or **ESC** to quit

## About

VibeOS is a hobby operating system with a retro Mac aesthetic.

Built with vibes by humans and Claude.

### History

1. Started as a bare-metal ARM64 kernel
2. Added framebuffer and console
3. Built a windowed desktop
4. Now running Python!
